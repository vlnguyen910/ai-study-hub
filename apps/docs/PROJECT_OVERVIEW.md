# AI Study Hub - Project Overview

## Summary
AI Study Hub is a web platform that centralizes students' learning documents and enables fast retrieval and AI-assisted Q&A. It addresses fragmented storage, poor organization, and the difficulty of quickly finding or asking about previous materials.

## Motivation and Problems
Students typically store learning documents across multiple channels (e.g., cloud drives, messaging apps, social groups, email, and personal devices). This creates several issues:

- Documents are scattered across sources.
- Old materials are difficult to locate.
- There is no consistent classification system.
- Sharing remains manual and inefficient.
- Local storage is limited.
- It is hard to ask quick questions about stored content.

## Goals
The system is built to:

- Centralize document management for students.
- Reduce document loss and duplication.
- Improve search, access, and retrieval speed.
- Provide a convenient sharing environment.
- Integrate an AI chatbot for learning support.
- Support a practical fullstack development workflow.

## Primary Actors

- Guest
- User
- Admin
- ChatbotService

## Functional Requirements

### 1. Authentication

- Register an account
- Sign in
- Sign out
- Forgot password
- Update profile

### 2. Document Management

- Upload documents
- View document list
- Download documents
- Delete documents
- Edit document metadata
- View document details
- Search documents
- Filter documents by subject

### 3. Cloud Storage

- Upload files to cloud storage
- Show upload status
- Preview files

### 4. AI Chatbot

- Chat with the chatbot
- Ask questions about documents
- Receive AI-generated answers
- View chat history
